# Gerador de Recibos em Python

Sistema desktop desenvolvido em Python para geração automática de recibos em PDF.

## Tecnologias
- Python
- Tkinter
- ReportLab
- Pillow

## Funcionalidades
- Interface gráfica
- Geração de PDF
- Campos editáveis
- Exportação automática

## Instalação

```bash
pip install -r requirements.txt
